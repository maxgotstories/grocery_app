# Grocery Store Web App (PHP/MySQL)

A lightweight CRUD application built to manage grocery lists with real-time status tracking and date-based filtering.

## 🚀 Features
* **Full CRUD:** Create, Read, Update, and Delete grocery items.
* **Status Tracking:** Categorize items as Pending, Bought, or Not Available.
* **Date Filtering:** View grocery lists for specific dates.
* **Responsive UI:** Built with Bootstrap 4 for mobile and desktop compatibility.

## 🛠️ Tech Stack
* **Backend:** PHP 8.x
* **Database:** MySQL
* **Frontend:** HTML5, CSS3, Bootstrap 4

## ⚙️ Setup Instructions
1. Clone the repo into your XAMPP `htdocs` folder.
2. Open PHPMyAdmin and create a database named `grocerydb`.
3. Import the `database.sql` file provided in this repo.
4. Ensure Apache and MySQL are running in XAMPP.
5. Visit `http://localhost/grocery_app/index.php`.
